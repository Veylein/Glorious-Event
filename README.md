This project is a responsive, elegant landing site for Kacy Thomas, event planner.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

3. Build:

```bash
npm run build
```

4. Deploy to GitHub Pages (set `homepage` in package.json or repository settings):

```bash
npm run deploy
```

Notes

- Update portfolio images and items in `content/portfolio.json` to change showcased events without touching React code.
- Update form endpoint in `src/components/Contact.jsx` to your Formspree or serverless form handler.
- Fonts are loaded from Google Fonts in `index.html`.
